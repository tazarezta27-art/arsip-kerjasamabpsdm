🔗 Tambah Link Google Drive

📋 Cara mendapatkan link Google Drive:
1. Buka file di Google Drive
2. Klik kanan pada file → pilih Bagikan
3. Ubah akses ke "Siapa saja yang memiliki link"
4. Klik Salin link → paste di bawah

💾 Backup & Restore Data

Export Backup

Simpan semua data dokumen, akun, dan link GDrive ke file .json di komputer Anda.

✅ Backup berhasil diunduh!
Simpan file ini di tempat yang aman.

Import / Restore

Muat data dari file backup .json sebelumnya. Data saat ini akan digantikan.

✅ Data berhasil dipulihkan!
Halaman akan dimuat ulang.
⚠️ Penting: Data tersimpan di browser (localStorage). Jika browser di-reset, data browsing dihapus, atau dibuka di komputer/browser lain, data bisa hilang. Lakukan backup rutin dan simpan file .json-nya di Google Drive atau flashdisk.
Data yang disimpan dalam backup: Semua dokumen PKS & Nota Kesepahaman, link Google Drive, dan daftar akun pengguna.
